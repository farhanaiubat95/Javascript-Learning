1.The let and const keywords were added to JavaScript in 2015.
2.The var keyword should only be used in code written for older browsers.
3.JavaScript identifiers are case-sensitive.
4.It's a good programming practice to declare all variables at the beginning of a script. 5.Variables declared with the var always have Global Scope. { var x = 2; }


What is Good?

  1. let and const have block scope.
  2. let and const can not be redeclared.
  3. let and const must be declared before use.
  4. let and const does not bind to this.
  5. let and const are not hoisted.
